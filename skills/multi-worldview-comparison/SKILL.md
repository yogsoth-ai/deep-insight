# Multi-Worldview Comparison

See what single-perspective analysis misses.

## Operations

catwoe-analysis → reframing-matrix → appreciative-discovery (optional)

## Available SOPs

**Subagent:** catwoe-analysis, reframing-matrix, appreciative-discovery
**Shared:** multi-stakeholder-simulation

## Execution Guidance

Apply CATWOE (Customers, Actors, Transformation, Weltanschauung, Owner, Environment) from at least 4 stakeholder perspectives, then use reframing matrix (engineer, social scientist, artist, economist or domain-appropriate equivalents) to see what each profession would focus on.

Key insight: the problem looks fundamentally different from each perspective. The "real" problem is the intersection of all perspectives, not any single one.

## Minimum Yield

```
<HARD-GATE>
- CATWOE analyses completed: >= 3 perspectives
- Reframing matrix perspectives: >= 4
- Overlooked framings identified: >= 2
- Synthesis of cross-perspective insights: >= 1
</HARD-GATE>
```
